Pengertian Link

HTML link atau hyperlink, adalah sekumpulaan kata ataupun gambar yang apabila di-klik akan menuju ke dokumen/halaman lain atau bagian lain dari dokumen/halaman.

Cara Membuat link di HTML

Tujuan kata Hypertext dari HTML adalah membuat sebuah text yang ketika di-klik akan pindah ke halaman lainnya. HTML menggunakan tag untuk keperluan ini.

Link ditulis dengan yang merupakan singkatan cari anchor (jangkar). Setiap tag setidaknya memiliki sebuah atribut href. Dimana href berisi alamat yang dituju (href adalah singkatan dari hypertext reference).

Macam-macam Link di HTML :

  1. Link Relative
  2. Link Absolute
  3. Link Kebagian lain dalam dokumen yang sama
  4. Link ke e-mail

1. Link Relative

Link relative dibuat apabila kita membuat suatu link pada page yang lain (filenya beda) pada komputer yang sama (web yang sama), tidak memerlukan alamat internet yang lengkap. Contoh kita akan membuka file hasil latihan list kita minggu lalu.Contoh lain kita akan menuju file di folder yang berbeda

2. Link Absolute

Link absolute dibuat apabila akan membuat link ke page web yang berada di luar website kita. Sehingga alamat yang akan ditulis harus lengkap sesuai alamat web yang akan dituju. Contoh kita akan membuka web detik.com dan Google

3. Link ke e-mail

Link e-mail digunakan untuk membuat link saat kita akan membuat/mengirim suatu e-mail (tidak akan bisa digunakan jika aplikasi email client tidak terinstal). Contoh kita akan mengirim email ke alamat ccdplayerx@gmail.com

4. Link ke bagian lain di halaman yang sama

Link ini digunakan apabila page/halaman web yang ada terlalu panjang sehingga kita harus melakukan scrool untuk menuju bagian yang kita inginkan, untuk mempermudah kita gunakan link dokumen ke bagian lain dari halaman yang sama, yang harus kita lakukan adalah :

  • Membuat id (name) pada bagian yang dituju
  • Membuat script Link pada tulisan yang akan kita tuju

Misal kita akan membuat link bagian dokumen di latihan web ini yaitu bagian atas jenis-jenis link, silahkan pelajari scriptnya
